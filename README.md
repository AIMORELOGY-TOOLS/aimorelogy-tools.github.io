# AIMORELOGY 用户管理后台

这是一个现代化的用户管理后台系统，专门为 AIMORELOGY 微信登录系统设计。提供直观的用户界面来管理用户数据、查看统计信息和监控系统状态。

## 🌟 功能特性

### 📊 仪表盘
- **实时统计**: 总用户数、VIP用户、SVIP用户、今日活跃用户
- **用户等级分布图**: 直观显示各等级用户占比
- **注册趋势图**: 最近7天的用户注册趋势
- **自动刷新**: 支持自动数据刷新，保持信息实时性

### 👥 用户管理
- **用户列表**: 分页显示所有用户信息
- **搜索功能**: 支持按昵称或OpenID搜索用户
- **等级筛选**: 按用户等级筛选显示
- **用户详情**: 查看用户完整信息和使用统计
- **编辑用户**: 修改用户昵称和等级
- **删除用户**: 安全删除用户数据
- **数据导出**: 导出用户数据为CSV格式

### 📈 数据分析
- **使用统计图**: 按使用次数分组的用户分布
- **活跃度雷达图**: 各等级用户的平均活跃度对比
- **多维度分析**: 深入了解用户行为模式

### ⚙️ 系统设置
- **API配置**: 自定义后端API地址
- **刷新间隔**: 设置自动刷新时间间隔
- **系统状态**: 实时监控API连接状态
- **版本信息**: 显示系统版本和更新时间

## 🚀 快速开始

### 1. 部署到GitHub Pages

1. **Fork或Clone仓库**
   ```bash
   git clone https://github.com/Jeff010726/AIMORELOGY-TOOLS-BACKSTAGE.git
   cd AIMORELOGY-TOOLS-BACKSTAGE
   ```

2. **推送到GitHub**
   ```bash
   git add .
   git commit -m "初始化用户管理后台"
   git push origin main
   ```

3. **启用GitHub Pages**
   - 进入仓库设置页面
   - 找到 "Pages" 选项
   - 选择 "Deploy from a branch"
   - 选择 "main" 分支
   - 点击 "Save"

4. **访问后台**
   - 访问: `https://your-username.github.io/AIMORELOGY-TOOLS-BACKSTAGE/`

### 2. 本地开发

1. **启动本地服务器**
   ```bash
   # 使用Python
   python -m http.server 8080
   
   # 或使用Node.js
   npx serve .
   
   # 或使用Live Server (VS Code扩展)
   ```

2. **访问本地后台**
   - 打开浏览器访问: `http://localhost:8080`

## 🔧 配置说明

### API配置
后台系统需要连接到微信登录系统的后端API。默认配置：

```javascript
// 默认API地址
const API_BASE_URL = 'https://aimorelogybackend.site';

// 支持的API端点
- GET  /health                    // 健康检查
- GET  /admin/list_all_keys      // 获取所有KV键
- GET  /admin/get_user?key=xxx   // 获取用户数据
- POST /admin/update_user        // 更新用户信息
- DELETE /admin/delete_user      // 删除用户
```

### 自定义配置
在系统设置页面可以修改：
- **后端API地址**: 如果你的后端部署在其他域名
- **数据刷新间隔**: 10-300秒之间，默认30秒
